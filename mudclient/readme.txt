Tortilla Mud Client
Клиент для многопользовательских онлайновых текстовых игр в жанре MUD
Системные требования: Windows XP и выше, Linux (через Wine).

О клиенте можно прочитать:
- На сайте программы: https://tmud.github.io/
- В группе Вконтакте, посвященной мадам: http://vk.com/club92727293
- В справке к клиенту программе (в меню ?/Справка).
- В описании к клиенту (в меню ?/О программе).

Важно !!!
------------------------------------------------------------------------------------------------------------------------------------------
1) Включайте режим га/автозавершения в игре (см. команду режим в маде). Он необходим для полноценной работы клиента
и плагинов. Если сервер не поддерживает данный режим, необходимо написать шаблон prompt-cтроки мада. По шаблону prompt
читайте справку к клиенту. Без данного режима клиент и плагины в некоторые моменты игры могут работать некорректно или
совсем не работать.

2) Немного изменился формат триггеров. В игре существуют 2 типа строк:
- незавершенные строки (где просят ввести чтото - имя пользователя, логин, пароль и т.д.)
- завершенные строки (все остальные игровые строки).
Клиент не умеет различать такие строки (если сам сервер допускает незавершенные строки). Поэтому для триггеров сделан
отдельный режим работы для незавершенных строк.
Чтобы триггер срабатывал на любую часть строки (режим для незавершенных строк), нужно в конце шаблона триггера указать знак $.

Особенности:
- Такой режим сделан только для триггеров.
- Все остальные элементы (подсветки, замены и т.д.), а также триггеры без $ в конце шаблона, работают только с завершенными
строками.
- Необходимо указывать строку для такого триггера полностью, чтобы избежать ложных срабатываний.
- Не рекомендуется использовать переменные для исключения ложных срабатываний.

Например:
#action {^Введите пароль:$} { abracadabra }

Все это необходимо, чтобы решить проблему с разрывами строк в клиенте.

Мини FAQ
------------------------------------------------------------------------------------------------------------------------------------------
1) Чтобы подключиться к маду в профилях по умолчанию, которые идут с клиентом, нужно нажать F12.
2) Чтобы отключить сообщения об изменениях переменных, триггеров и др. нужно выполнить команду #message all off
3) Чтобы отключить сообщения от плагинов ([plugin] ... ) нужно в Настройках/Общие настройки снять галочку.
4) Чтобы отключить дублирование системных команд нужно в Настройках/Общие настройки снять галочку.
