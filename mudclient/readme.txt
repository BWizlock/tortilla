Tortilla Mud Client
Клиент для многопользовательских онлайновых текстовых игр в жанре MUD.

Системные требования:
- Windows XP и выше
- Linux, MacOS (с помощью Wine).

О клиенте подробно прочитать, найти материалы, плагины и другое:
- В FAQ клиента (справка), вводить в комадной строке клиента:
  -- #help
  -- #help faq
  -- #help triggers
  -- #help features
  -- #help capabilities
  -- #help message
