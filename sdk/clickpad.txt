Плагин clickpad обладает своим API.

bool set(row,column, table) - установка новых параметров кнопки. table - параметры, см. ниже.
table get(row, column) - получение параметров кнопки.
bool update(row, column, table) - обновление отдельных параметров для кнопки.
bool clear(row, column) - удаление кнопки
bool show(rows, columns) - задать сколько рядов и колонок с кнопками показывать.

table - Lua-таблица с параметрами кнопки.
table = {
  text - текст на кнопке (строка),
  command - команда кнопки (строка),
  tooltip - всплывающая подсказка на кнопке,
  image - картинка на кнопке (строка), путь к файлу с картинкой,
  imagex, imagey - координаты картинки на атласе (целочисленные),
  template - флаг, что это шаблон (булевский тип), шаблон - команда без запуска, т.е. команда копируется в командную строку.
}
